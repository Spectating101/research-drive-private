```text
Home · Library · Discover · Synthesis · Resources · Profile · Settings
```

```text
Navigation | Centre | Detail / Ask
```

For active research objects:

```text

Detail = meaning + current decision
Ask = intelligence + supported operation
Backend = durable consequence
Centre = consequence becomes visible
```

No casual new permanent destinations or second workspaces.

### Library

`docs/LIBRARY_FULL_SCALE_FREEZE_2026-07-15.md`

Binding thesis:

```text
GOOGLE DRIVE'S LOCATIONAL CONFIDENCE
+
A RESEARCH EVIDENCE ESTATE'S SEMANTIC / AUTHORITY TRUTH
```

Core row:

```text
TITLE                 SOURCE          VERIFY           STATE
plain-language description
```

Source, verification, and readiness are independent axes.

Collection selection changes research location.

Asset selection inspects evidence while preserving location.

Collections are research organisational contexts, not physical archive directories.

One durable asset may belong to multiple collections without duplication.

The right rail is Composer / Cite-Agent operating the exact selected collection or evidence asset through the stable `DETAIL | ASK` grammar.

Library remains the strongest complete product surface.

### Home

`docs/HOME_FULL_SCALE_FREEZE_2026-07-16.md`

Approved Iteration 10 composition:

```text
PICK UP                         RESOURCE HEADROOM

        ↓

RECOMMENDED EVIDENCE

        ↓

RECENT TRAIL
```

Home is a compact research re-entry desk.

It is not a command centre and not a sustained research work surface.

Frozen functions:

```text
PICK UP
recover exact resumable work state

RESOURCE HEADROOM
show 1 pinned + 1 material bounded resource

RECOMMENDED EVIDENCE
show grounded evidence suggestions

RECENT TRAIL
show durable material machine consequences
```

Hard desktop caps:

```text
PICK UP
1 primary + 1 secondary max

RESOURCE HEADROOM
2 resources max

RECOMMENDED EVIDENCE
2 recommendations max

RECENT TRAIL
3 consequences max

DESKTOP PAGE SCROLL
NEVER
```

Resource bars require:

```text
used
+
real cap / denominator
+
current authoritative state
```

Reset / renewal wording requires real reset authority.

Recommendations require grounded recommendation authority.

Recent Trail excludes browser history and ordinary poll / worker noise.

Home exact resume points activate the underlying typed object. Centre shows object + resumable state; Detail explains exact object truth and decision; Ask operates the same object.

```text
Agent page
Activity resurrection
acquisition workspace
pipeline builder
Evidence Builder
Semantic / AI / Advanced Search tabs
Research Query / Browse / Source Finder modes
worker-dashboard History
infinite lifecycle feed
new permanent route for review states
```

Do not casually rename permanent tabs.

Discover tabs remain exactly:

```text
Explore | History
```

Right rail tabs remain exactly:

```text
DETAIL | ASK
```

Visible intelligence identity may expose Composer / Cite-Agent inside Ask where truthful, but the application grammar does not become `Detail | Composer`.

Do not treat LLM inference as durable product authority.

```text
MODEL PROSE
≠
SOURCE AUTHORITY

MODEL SIMILARITY
≠
COLLECTION MEMBERSHIP

MODEL INFERENCE
≠
PROFILE FACT

MODEL SUMMARY
≠
VERIFICATION RECORD
```

Durable consequences must write to supported system state before the centre presents them as truth.
